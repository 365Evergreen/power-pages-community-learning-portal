# Power Pages SPA — Frontend (Vite + React)

Quick start

1. cd frontend
2. npm install
3. npm run dev

Notes

- This scaffold uses React + Vite.
- Replace the `deploy` script in `package.json` with your Power Platform CLI (pac) or CI deployment commands to export/pack the SPA as a web resource.
- For Power Pages, build output lives in `dist/` after `npm run build`.
